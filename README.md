# German Writing Coach 🇩🇪

독일어 학습자를 위한 플래시카드 웹 애플리케이션

## 프로젝트 개요
독일어-한국어 단어 플래시카드로 어휘를 학습하고,
알았다/몰랐다 체크를 통해 학습 진행률을 확인할 수 있습니다.

## 주요 기능
- 단어 카드 뒤집기 (독일어 ↔ 한국어)
- 알았다 / 몰랐다 체크
- 학습 진행률 표시
- C1 레벨 단어 데이터 제공

## 사용 기술
- HTML / CSS / JavaScript
- 별도 서버 없이 브라우저에서 실행
